# ألواح الرافدين - Mesopotamian Glyphs

العراق ليس بلداً، إنه وعاء الزمن.

## عن اللعبة
لعبة روحانية عراقية من 7 مستويات، تروي حكاية العراق عبر العصور.
كل لوح هو مرحلة، وكل بصمة هي رسالة من جدك.

## المستويات
1. وركاء - حيث بدأ كل شيء
2. أهوار الجنوب - الماء الذي يكتب
3. بغداد المستديرة - مدينة السلام
4. سامراء - المئذنة الملوية
5. كربلاء - وردة على الرمل
6. بابل - بوابة لا تذهب لأحد
7. العراق في قلبي - الذاكرة

## كيفية اللعب
- اضغط على اللوح لترك بصمتك الذهبية
- اكمل العدد المطلوب من البصمات لكل مستوى
- شاهد الحرف الكوفي يبكي عند الإكمال
- في النهاية، ودّع الوشاح الأبيض وهو يطير

## تقنيات مستخدمة
- HTML5
- CSS3 (مع تأثيرات 3D)
- JavaScript (Web Audio API)

## مطورة مجاناً
هدية لكل عراقي يحن لوطنه
